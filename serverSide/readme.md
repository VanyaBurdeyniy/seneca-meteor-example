# Seneca meteor example server

## How to run server

Make your **config/config.json** file
Run: 
``` bash
npm i
npm start
```


## config/config.json example

``` javascript
{
    "server": {
        "port": 3000
    },
    "log": {
        "path": "./logs/log.txt"
    }
}
```

## API Documentation

You can generate the docs by running

``` bash
npm run docs
```
